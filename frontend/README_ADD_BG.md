To add a custom background image you provided:

1. Save your photo to: `frontend/src/assets/bg.jpg`
   - Use a reasonably large image (e.g., 1920x1080) for best cover results.
2. Start the dev server (Vite) and the app will automatically use that image as the full-page background.

Notes:
- The CSS applies a subtle blur and darkening for readability.
- If you want the image loaded from an external URL instead, edit `frontend/src/index.css` and replace the `url('/src/assets/bg.jpg')` with the external URL.
